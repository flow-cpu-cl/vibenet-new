# 🚀 VIBE NET - HTML Version

Jednoduché HTML, CSS a JavaScript řešení pro web VIBE NET.

Bez složitého setupu - jen HTML soubor!

---

## 📋 Co je součástí?

- ✅ `index.html` - Kompletní design s Tailwind CSS
- ✅ Všechny animace a efekty
- ✅ Responsive design (mobil + desktop)
- ✅ Tailwind CSS inline (žádné dalších dependenciích)

---

## 🚀 Deploy na Vercel

### Krok 1: GitHub
1. Vytvořte nový projekt na GitHub
2. Pushujte soubory
3. Vercel se automaticky nasadí!

### Krok 2: Vercel
1. Jděte na https://vercel.com
2. Import from GitHub → váš projekt
3. Deploy!

**Hotovo!** Web je LIVE! 🎉

---

## ✏️ Jak editovat obsah?

Otevřete `index.html` v editoru (VS Code, atd.):

### Změnit text:
```html
<h1>Internet Without The Leash.</h1>  ← Změňte sem
```

### Změnit cenu:
```html
<span class="font-display text-5xl">48 CR</span>  ← Změňte tady
```

### Změnit barvu tlačítka:
```html
<button class="bg-[#0071e3]">  ← Změňte kód barvy
```

---

## 🎨 Tailwind CSS

Projekt používá **Tailwind CSS** přes CDN - žádná instalace není potřeba!

Pokud chcete editovat styly, podívejte se do `<style>` v HTML.

## 💡 Editace spolu

Kdykoliv chcete cokoliv změnit:
1. Otevřete `index.html`
2. Najděte text co chcete změnit
3. Upravte
4. Uložte (Ctrl+S)
5. Push na GitHub
6. Vercel se automaticky nasadí

---

## 🎯 Sekce k editaci:

- **Hero Section** (nadpis, text) - řádky ~85-110
- **Features** (ceny, popisy) - řádky ~180-250
- **Pricing** (cenové plány) - řádky ~350-450
- **Why Section** (benefity) - řádky ~500-550
- **Specs** (technické info) - řádky ~600-650
- **Footer** (kontakt, sociální) - řádky ~680-720
